# Changelog

All notable changes to the **Shinobi Black** theme will be documented in this file.

## [1.0.0] — 2026-07-12

### Added

- Pure OLED black workbench colors covering all major UI surfaces
- Comprehensive syntax highlighting for C/C++, Python, Java, JavaScript, TypeScript, HTML, CSS, JSON, Markdown, and SQL
- Full ANSI terminal color palette
- Bracket pair colorization support
- Diff and merge editor colors
- Git decoration colors
- Debug toolbar and console colors
- Peek view colors
- Notification center colors
- Minimap colors
- Scrollbar styling

### Color Palette

- Background: `#000000`
- Foreground: `#EAEAEA`
- Keywords: `#F14E95`
- Types: `#4BC6F7`
- Functions/Strings: `#B277FA`
- Numbers: `#FFCB6B`
- Comments: `#8B949E`
- Booleans: `#C3E88D`
- this/self: `#FFA500`
- Operators: `#89DDFF`
